# Rock Paper Scissor Game (Python)

## Description
This project is a two-player Rock Paper Scissor game developed using Python.
The program allows players to play multiple rounds, validates user input,
keeps track of scores, and displays the final winner.

## Requirements
- Python 3.x

## How to Run the Program
1. Download or clone this repository
2. Open terminal or command prompt
3. Run the Python file using:
   python rock_paper_scissor.py

## Features
- Two-player gameplay
- Input validation
- Score calculation
- Final result display
